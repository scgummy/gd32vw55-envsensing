# iceasy-eval-gd32vw55

GD32VW55-IOT 评测

## 食用方法

将官方的 Wi-Fi & BLE SDK 压缩包 [`GD32VW55x_RELEASE_V1.0.3a.zip`][sdk] 解压到 `/vendor` 目录下，
注意要脱去外层目录，即最终的目录结构应该类似

```
/vendor
├── config
├── docs
├── MBL
├── MSDK
├── release_notes.txt
├── ROM-EXPORT
└── scripts
```

[sdk]: https://gd32mcu.com/cn/download/7?kw=GD32VW5
